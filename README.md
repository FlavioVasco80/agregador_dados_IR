# Agregador de Dados para Imposto de Renda - Friendly Lion

**DESCRIÇÃO DO PROJETO**

O Friendly Lion é uma ferramenta robusta feita em Excel para centralizar e validar informações necessárias para o Imposto de Renda. O foco foi transformar a coleta de dados, muitas vezes confusa, em um processo linear e seguro.

**CONTEXTO PROFISSIONAL**

Aplicando minha experiência em gestão de processos e com orientação do professor Felipe Aguiar, estruturei esta ferramenta para garantir integridade dos dados e facilitar o fluxo de preenchimento, reduzimdo erros comuns na organização de informes bancários e notas fiscais.

**FUNCIONALIDADES TÉCNICAS** 

Navegação Customizada: Menu interativo via botões para trânsito entre abas (UX). 

Validação de Dados: Campos estruturados para evitar preenchimentos incorretos em CPFs, telefones e valores. 

Consolidação de Rendimentos: Soma automática de informes de múltiplos bancos com anexos referenciados. 

Histórico de Entradas: Aba de notas e holerites para controle mensal de receitas.

**TECNOLOGIAS UTILIZADAS**

Microsoft Excel: Menus interativos, hiperlinks, fórmulas de agregação e validação. 

Git/GitHub: Para o controle da versão, documentação e portifólio (README) do projeto.
